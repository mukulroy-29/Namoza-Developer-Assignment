## Project Overview

This project is a responsive healthcare landing page developed for **OrthoNow Orthopaedic Clinic** as part of the Namoza Developer Assignment.

The landing page is designed to generate consultation bookings while following modern UI/UX, accessibility, SEO, and conversion optimization best practices.

---

- Responsive Design
- Mobile-First Layout
- Sticky Navigation
- Sticky Mobile CTA
- Trust Indicators
- Patient Testimonials
- FAQ Section

- Success Confirmation Screen

- Accessibility Best Practices

- Fast Loading Performance

## Google Tag Manager Event

The following Data Layer event is triggered after successful form submission:

```javascript
window.dataLayer.push({
event: "consultation_form_submitted",
lead_source: "Google Ads",
clinic_region: "Bengaluru",
form_type: "minimal_booking",
submission_time: new Date().toISOString()
});
```
